---------------------------------------------> FONCTIONNEMENT DU PROGRAMME <------------------------------------------

Le programme, en se basant sur les listes, vous permet de pouvoir calculer les moyennes de vos etudiants, faire le classement. et le programme se charge d'enregistrer votre classement dans un fichier texte "classement.txt" qui se sera dans le dossier d'exécution du programme.

A l'ouverture du programme un menu apparait avec comme option 1.nouveau calcule, 2.charger un jeu de donnée, 3.quiter.


========================================> Explication des options du menu<============================================


la première option vous permettra de faire un nouveau calcule de moyenne, le classement et l'enregistrement dans le fichier "classement.txt" est automatique après saisie des données initiales(notes des etudiants).

NB: c'est le programme qui cré le fichier "classement.txt" pour y enregistrer les données.


LA deuxième option vous permet d'afficher dans votre terminal le classement précédant.

NB:pour cette option vous devriez avoir déjà effectué un premier classement.

La dernière option permet de fermer le programme.
